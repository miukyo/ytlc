# ytlc

Unofficial JavaScript/TypeScript library for reading YouTube live chat via InnerTube.

## Install

```bash
npm install ytlc
```

## Build From Source

```bash
npm install
npm run build
```

## Generate API Site (TypeDoc)

```bash
npm run docs
```

Generated output: `docs/reference`

## Quick Start

Look at reference docs for how to use it. https://ytlc.pages.dev/

## Notes

- This is an unofficial parser over YouTube internal payloads. Schema may change anytime.
