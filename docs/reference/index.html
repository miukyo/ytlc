    ytlc

    Unofficial JavaScript/TypeScript library for reading YouTube live chat via InnerTube.

    Install

    npm install ytlc

    Build From Source

    npm install
    npm run build

    Generate API Site (TypeDoc)

    npm run docs

    Generated output: docs/reference

    Quick Start

    import { YTLiveChat } from "ytlc";

    const chat = new YTLiveChat({ requestFrequency: 1000 });

    chat.on("initialPageLoaded", ({ liveId }) => {
    	console.log("Monitoring", liveId);
    });

    chat.on("chatReceived", ({ chatItem }) => {
    	const text = chatItem.message
    		.map((part) => (part.type === "text" ? part.text : part.alt ?? part.url))
    		.join("");

    	console.log(`${chatItem.author.name}: ${text}`);
    });

    chat.on("errorOccurred", ({ error }) => {
    	console.error(error.message);
    });

    await chat.start({ handle: "@YouTubeChannelHandle" });

    Dummy API

    Use sendDummy() directly on YTLiveChat when building UI/tests without real network calls:

    import { YTLiveChat } from "ytlc";

    const chat = new YTLiveChat();
    chat.on("chatReceived", ({ chatItem }) => {
    	console.log(chatItem.author.name, chatItem.message);
    });

    chat.sendDummy({ mode: "text", text: "hello from dummy", authorName: "local-test" });

    Auto reconnect is enabled by default. You can tune it with:

    • autoReconnect
    • reconnectMaxAttempts
    • reconnectDelayMs
